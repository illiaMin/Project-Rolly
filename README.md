\# Project Rolly



A top-down 2D action game about obedience, participation, and refusal,

set inside a functioning robot factory.



\## Core Idea

The player controls a robot used as a tool by another entity.

Progression rewards efficiency and compliance.

Only refusal can break the cycle.
